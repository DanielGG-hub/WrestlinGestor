# \# WrestlinGestor

# 

# Aplicación de escritorio desarrollada en Java con Swing para el apartado gráfico y MySQL como base de datos.

# Permite gestionar una base de datos empresarial mediante operaciones CRUD.

# 

# \---

# 

# Funcionalidades

# 

# \# Botón gestionar proyectos

* # Crear, editar, eliminar, buscar proyectos.
* 



# 

# \# Gestión de la tabla clientes

* # Crear, editar, eliminar, buscar clientes.

# 

# \# Gestión de la tabla Empleados

* # Crear, editar, eliminar, buscar empleados.

# 

# \# Gestión de la tabla Pagos

* # Crear, editar, eliminar, buscar, ver solo proyectos pagados, ver solo proyectos sin pagar.

# 

# \# Gestión de la tabla Contratos

* # Crear, editar, eliminar, buscar, ordenar por fecha final del contrato.

# 

# \# Gestión de la tabla Empleados Proyecto

* # Nuevo, editar, eliminar, buscar proyecto.
